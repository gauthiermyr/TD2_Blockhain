# TD2 - Bitcoin
##vStart

run `python td2.py` to run the CLI application

## Commands
### Menu

 - `1`: Generate a valid mnemonic phrase
 - `2`: Import a mnemoic phrase
 
### App

- `1`: Display Master Private Key & Chain Code 
- `2`: Display Master Public key 
- `3`: Exit the app
